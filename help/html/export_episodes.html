Export Episodes
You can export the episodes of a series either by right clicking on a series on the [Series Panel] and choose export episodes of... or by clicking Tools-Export episodes of...
Then a file browser window opens to select the file to save the episodes to.
The default file name is [NAME_OF_SERIES] S[SEASON].eef .
The file is a tab delimeted text file with one episode per line and the format of each line is:
EpisodeNumber(integer) TAB Episode title (String) TAB Aired date (YYYY-MM-DD)
If the date is empty the current date is used.
