Date Format Patterns
Day (pattern : d for numeric , E for textual)
The number of pattern letters are the minimum number of digits.Shorter numbers are zero padded (eg d -> 4 , d -> 14 , dd -> 04 , dd -> 14 , ddd -> 004 etc
If the number of pattern letters are less than for a short form is used (eg EEE -> Tue), else the full form is used (eg EEEE -> Tueaday)
Month (pattern : M)
If the number of pattern letters is 1 or 2 the numeric represantation of the month is used (zero padded if needed) (eg M -> 5 , M -> 11 , MM -> 05, MM -> 11).If 3 pattern letters are provided the textual short form is used (eg MMM -> Jul , MMM -> Aug etc) else the full form is used (eg MMMM -> August, MMMM -> December etc)
Year (pattern : y)
if the number of pattern letters is 2, the year is truncated to 2 digits (eg yy -> 08 for 2008) if the numbers are 3 the 2digit year is zero padded (yyy -> 008 for 2008) . If the number is 4 The full year is used (yyyy -> 2008). For greater numbers the full form is zero padded
