Renaming episodes
Renaming episodes panel
The renaming episodes panel consists of an options panel and the files table.
In the options panel you can check / uncheck all episodes for renaming and you can select the separators between series title and season (season separator), the season and the episode number (episode separator) and between the episode and the title (title separator). Applying the changes you get a preview of the renaming in the files table.
The files table has 4 columns
  1. The id column
  2. The old name column
  3. The new name column
  4. Rename or not the episode (This column is disabled if the new name is the same as the old name)
After renaming you get a report for the number of files that renamed or not renamed.
