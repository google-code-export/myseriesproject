The Episodes Panel PopUp Menu
Enabled popup menu Disabled popup menu
When right clicking on the [Episodes Panel] the episodes popup menu is shown. If the clicking is on an episode row the enabled popup menu is shown else the disabled one.
The following menu items are available:
  1. Add New Episode:Brings up the [Add new Episode Panel] for adding a new episode of the current series in the database.
  2. Delete Episode:Deletes the current episode from the database or multiple episodes if multiple episodes are selected.
  3. View Episode:Opens the selected episode in the default media player application or the one you have provided in [General Options]. The path to the local folder that contains the episodes must be provided in [Add/Edit Series Panel]
  4. Download Episode subtitlesIf the www.tvsubtitles.net or the www.subtitleonline.com series code is provided (in [Add/Edit Series Panel]) you can search for subtitles in one of these webpages.
    If the primary language (set in [Internet options]) is not found the application will try to download the secondary one.
    If it's found and the local directory of the series is provided the subtitle file will be extracted there (also renamed according to the renaming options in [Renaming Options] if auto renaming is enabled).
    If multiple subtitles are found you will prompt to choose which subtitle to download.
    If downloaded from tvsubtitles.net the users' ratings of the subtitles are provided next to each title to help you avoiding the downloading of wrong/bad subtitle files.
    If the local directory is not provided or direct access to subtitle file is denied by the webpage server the default internet browser will open and download the file.
    You can also move downloaded subtitles (compressed or not) to the Main Series Directory provided in [General Options] and whenever you choose [Update Files] from the tools menu the subtitles will be moved to the series directory and renamed (if auto renaming is enabled).
  5. Download episode torrent.Let you search and download torrent from
    1. EzTv webpage ([eztv download])
    2. Isohunt webpage([isohunt download]).
