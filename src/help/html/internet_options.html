Internet Options
The following options are available in Internet Options
  • Number of columns in feeds - You can select the number of columns shown in the feeds panel.
  • Update feeds on startup - If you want the feeds to be updated when MySerieS starts.
  • Update feeds frequancy - The frequency to auto update feeds.Setting it to 0 will deactivate the auto update.
  • Check for updates - If the application should check for updates on start up.
  • Proxy settings - If you are behind a proxy you can set the host and port by checking the Proxy check box and provide the http host address and port in the textfields.
