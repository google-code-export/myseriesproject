Performance Options
The following options are available:
  • Update memory indicator:
    How often the memory indicator on the toolbar will be updated (if it's visible). Setting this to 0 will remove the indicator from the toolbar.
  • Screenshots quality:
    The quality of the series screenshots on the main application panel and on the schedule.
  • Hide deleted series rating:
    When viewing the ratings tab only the active series stats will show.
  • Auto Update Files:
    Whether to move subtitles and video files from the main series directory or not
